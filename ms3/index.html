Create Date Menu

A date menu is a menu that overrides the day menu for a particular date that you set. A date menu has its own schedule and menus, but follows the same guidelines and philosphy as the day menu. You have an opening and closing time for that date and the menus in accordance.

To create a date menu, navigate to the company tab on the sidebar of the admin, select Locations from the drop-down, and then select the desired location for which you would like to create a date menu.

Once there, select the create date menu button at the top right of the page. A creation dropdown will appear, from there you will provide an opening time, a closing time, and the date you want to set the menu for. When your ready to create the date menu click the create date menu.

Once created your date menu will show up in your date menu list, from there you will see three dots on the right side of the date menu item on the list. Clicking the 3 dots will allow you to delete or edit your date menu, editing will allow you to change times or add/removes menus. Menus operate the same way a day menu operates, set the menu and time and you click add menu with time button. You can also delete a menu by clicking the red X next to the chosen time
