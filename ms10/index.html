    Create A rank

    Navigate to the loyalty section on the side bar then click manage there you will see all your current ranks and info about the loyalty program.

    At the top right you will see two options, the rightmost option will be to create a new rank, click add rank to open the dropdown, set the minimum points required to be in that rank, the points per dollar they earn, and then the color for that the rank, the color will be a hex color and will show up as badge on the app. You can navigate to the info icons to learn more about each item for a better understanding, once finished click create rank, you will be able to see the rank in your list.

    From there you can view a loyalty rank and you can click edit your rank to edit the details of the rank. A module will appear allowing you to change the rank's name, points per dollar, and hex color.

    Every craspa account comes with a default rank with a minimum points of 0, so that all newly joined customers have a beginning rank.

    If you create a new rank while you have active customers it will take 24-48 hours for all your customers to be reorganized based on the new minimums, for example if you had a previous minimum rank between 0 and 1,000 and you created a minimum rank at 500 your customers above five hundred would be adjusted to the new rank.

    Point Rule#

    The default point rule is set to one year, the point rule when changed will not apply to all customers, only new customers and restarted customers after their rank points expire, from there you can save your changes and then it will be applied. This applies to all ranks. Point Rule is used cycle out inactive customers or keep customer trying again ever point period. Even though points and rank reset, your customer and you are still able to see lifetime points. Thus allowing you to reward or filter customer with high or low lifetime points. Edit the point rule by clicking adjust point on the top right of the loyalty manage page.

    You can't change a rank's minimum points or create a new rank with the same minimum points as an existing rank. Its best to not create more than 5 ranks.
