Understanding Order Details#

On the order detail page, you'll see all the orders details. Most boxes are self explanatory, however this guide will clarify some things about the order detail. The customer description box is the box to tell you about the customers description if they choose a pickup option that requires a description. Descriptions follow the same format, Color & Object, and will show as green when provided. The pickup option box tells you the pickup option the customer chose for that order. Order lead box tells you what caused the customer to make an order, either a notification or organically. To view all the items in the order, view the item list at the bottom of the order detail page. Finally the View In Square, clicking the button will open the order in Square for you view in your Square admin.
