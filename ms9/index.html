Create A pickup option

Pickup options are tied to a location so that you can have a dynamic option for having different pickup options for different locations. Navigate to the bottom of location detail page on the admin to pickup option list, click create pickup option, there a popup will be shown to fill out pickup option info. Enter in a display name, this is shown to the customer when checking out through the app and selecting a pickup option, and a code name for your end, if you have shorthand names for pickup options. Then select if a description is required, a description required means that when a customer is ready to pickup they're order they will provide a description of themselves in the app. For example if you have curbside a customer will say they are here but you'll have to know which car they are in. So they can provide a description of what vehicle they are in, like red honda suv, or about their persons, female white jacket.

You can set if the pickup option is active or not, a pickup options active status determine if the pickup option is shown in app.

Then you can give a description about the pickup option which is shown along side the display. A description may be explaining how your customers should retrieve their meal once they have arrived. Then click create to create the pickup option, once created it will show up in the pickup options list, where you can view the pickup option and and manage its settings how you like. Click the 3 dots to access the pickup option details.
