Create Day Menu

Create a day menu, by navigating to the location detail page of the location you want to set a menu. Navigate to the schedule section of the locations detail, where you'll see a schedule object for each day of the week and at the right hand of each schedule is two buttons one to set the operating hours and the other for menus for that. Topically you'll need to set this once, because you have topically have the same menu for the same day for week. When we say "create a day menu" that means adding a menu to a certain day along with the time the menu starts.

To manage the menu for the day, select the "edit day menu" button on the day you want to adjust, from there a prompt will be shown prompting you to add a menu along with showing existing menus. To add a menu you click the menu select on the bottom box, there a list of menus allowed for the selected location will appear, then add a start time for the menu. When your ready to add the menu, click add menu with time. You can not create a menu that clashes with a existing menu start time for that day.

The way menus work is that you choose a menu, choose a start time and that menu will be shown until a new menu with a start time greater than the previous one is available. So if you have a breakfast menu you can set it at the start of the day and it will stay there until the next menu time begins. Then if you have a lunch menu after the breakfast menu but no other menus after lunch. The lunch menu will show until the next day and so on, if no other menus are superseding the lunch menu times. It's ok to set a menu at the zero hour or times before your opening because your customer will no be able to make an order until your operating hours but will still be able to view it.
