    Create A Menu

    Navigate to menus tab then select menus on that dropdown there you will see a list of all existing menus, to create a menu go to the top right hand and click create menu, there you can make a name for the menu and decide if it is active or not, once created you will be directed to the menu details page.

    From there you will be able to create a special for that menu and highlighted items along with setting the categories and the locations allowed to use that menu, doing this allows you to add this menu to a day or date menu. When selecting a menu for a day menu on a location detail page, a menu will not show if the location has not been set for that specific menu. To add and remove a location, click on the location select, there a dropdown will appear of all your imported locations. Click on a dropdown item to add that location and click on the x of an added item to remove it. When your done, click the save changes button to confirm the change.

    you can change the menu name by going to the top right and clicking change menu name, there a popup will prompt you to change the menu. You can also delete the menu from the option at the top right.

    To set categories, click the category bar, a drop down will appear from which you can select categories you want to add, click on a dropdown item and it will add that category to the menu. Clicking the x of an added category item will remove that category from the category list, when your finished adding and removing your categories, click the save changes button to save the changes.

    caution

    If a menu has been added to a day menu and you remove the location form the menu the location will still have access to the menu until you remove the menu from that location
