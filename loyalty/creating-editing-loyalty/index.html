How To Create/Edit A Loyalty Rank#

Navigate to the loyalty section on the sidebar then click manage there you will see all your current ranks and info about the loyalty program.

At the top right you will see two options, the rightmost option will be to create a new rank click add rank to open the dropdown. Set the minimum points required to be in that rank, the points per dollar they earn, and then the color for that rank, the color will show up as a badge on the app.

You can hover over the info icons to learn more about each item for a better understanding, once finished click create rank and you will be able to see the rank in your list.

Editing a rank#

From there you can view a loyalty rank and you can click the 3 dots on your rank to edit the details of the rank. A module will appear allowing you to change the rank's name, points per dollar, and hex color.

Warnings#

You can't change a rank's minimum points or create a new rank with the same minimum points as an existing rank. It's best to not create more than 5 ranks.

If you create a new rank while you have active customers it will take 24-48 hours for all your customers to be reorganized based on the new minimums, for example if you had a previous minimum rank between 0 and 1,000 and you created a minimum rank at 500 your customers above 500 would be adjusted to the new rank.
